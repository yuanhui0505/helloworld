# Pickleball Pro Scorekeeper

這是一個專為匹克球 (Pickleball) 設計的記分 Progressive Web App (PWA)。它支援賽程管理、語音播報比分，並可完全離線使用。

## ✨ 主要功能

*   **🏆 賽程管理**:
    *   建立多場比賽，記錄每場的對戰隊伍 (如 "小王/小明" vs "大華/小李")。
    *   歷史賽程列表，自動保存於本地裝置 (LocalStorage)。
*   **🔢 專業記分**:
    *   符合匹克球標準記分規則 (發球方得分 - 接球方得分 - 發球號碼)。
    *   支援單打/雙打規則 (1號發球者/2號發球者/Side-out)。
    *   比賽結束判定 (11分制 / 領先2分)。
*   **🗣️ 語音播報**:
    *   得分或換發球時自動語音朗讀比數 (繁體中文)，讓球員專注於場上。
*   **📱 PWA 支援**:
    *   可安裝於手機/電腦主畫面，像原生 App 一樣運作。
    *   支援離線使用 (Offline Support)。
    *   **螢幕常亮 (Wake Lock)**: 比賽期間防止螢幕自動關閉。

## 📂 檔案結構說明

| 檔案名稱 | 說明 |
| :--- | :--- |
| **`pickleball.html`** | **核心程式碼**。包含所有 HTML 結構、CSS 樣式與 JavaScript 邏輯 (記分規則、賽程管理、UI 互動)。 |
| **`service-worker.js`** | **PWA 核心**。負責快取檔案，讓 App 能在無網路環境下執行。 |
| **`manifest.json`** | **安裝設定檔**。定義 App 名稱、圖示、啟動模式等，讓瀏覽器識別其為可安裝應用程式。 |
| **`icon-192.png`** | 192x192 像素的 App 圖示 (主畫面用)。 |
| **`icon-512.png`** | 512x512 像素的 App 圖示 (啟動畫面/商店用)。 |

## 🚀 如何使用/安裝

由於本專案包含 Service Worker，建議需透過 HTTPS 或 Localhost 環境執行。

### 快速體驗 (需安裝 Python 可能)
1. 在專案目錄下打開終端機。
2. 執行 `python -m http.server 8000`。
3. 瀏覽器開啟 `http://localhost:8000/pickleball.html`。
4. 點擊瀏覽器網址列右側的「安裝」按鈕。

### 部署建議
建議上傳至 **GitHub Pages** 或 **Vercel** / **Netlify** 等靜態網頁託管服務，即可透過手機瀏覽器訪問並安裝使用。
